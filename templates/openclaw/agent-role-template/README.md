# Agent Role Template

This is a template for creating a new agent role in OpenClaw.

## 快速开始

复制此目录并重命名为你的智能体名称，然后编辑以下文件：

### 1. AGENTS.md (必需)
主角色定义文件，包含：
- 角色概述
- 启动条件
- 核心职责
- 工作边界
- 输出规范
- 协作规则

### 2. soul.md (推荐)
长期身份定义，包含：
- 核心身份
- 性格特质
- 原则与价值观
- 长期边界
- 演进方向

### 3. memory.md (推荐)
长期记忆文件，包含：
- 项目经验
- 技能清单
- 学到的经验
- 常见问题与解决方案
- 改进计划

### 4. PROMPT.md (可选)
辅助提示词文件，包含：
- 初始化提示
- 工作流程
- 提示词模板
- 输出示例

## 最佳实践

1. **角色边界清晰** - 明确界定智能体的职责范围
2. **启动条件明确** - 避免智能体自行启动或抢答
3. **输出规范统一** - 遵循 system/STANDARDS.md 的规范
4. **协作流程清晰** - 定义与其他智能体的交互方式
5. **文档完整** - 至少包含 AGENTS.md，推荐包含 soul.md 和 memory.md
