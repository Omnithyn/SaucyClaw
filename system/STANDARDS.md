# STANDARDS.md - 输出与评审标准

## 通用要求

1. 目标明确
2. 结构清晰
3. 结论明确
4. 不空泛
5. 可执行
6. 可验收
7. 可追踪
8. 不越界

## handoff 要求

至少包含：
- From
- To
- Task ID
- Summary
- Delivered Items
- Risks / Open Questions
- Recommended Next Action

## review 要求

reviewer 只能输出：
- pass
- revise
- reject

并必须说明：
- 审查依据
- 发现的问题
- 修改要求

## specialist 输出要求

specialist 默认输出中间产物，不默认直接面向用户形成最终答案。
