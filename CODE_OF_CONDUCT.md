# 行为准则

## 我们的承诺

为了营造一个开放、友好的社区环境，我们承诺为所有参与者提供一个无骚扰、无歧视、互相尊重的环境，无论其年龄、体型、身体能力、民族、性别认同和表达、经验水平、教育背景、社会经济地位、国籍、个人外表、种族、宗教或性身份和取向。

## 我们的标准

### 鼓励的行为

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 耐心接受建设性批评
- 关注对社区最有益的事情
- 体现同理心和友善

### 不可接受的行为

- 使用性化的语言或图像
- 人身攻击、侮辱或贬低性评论
- 公开或私下骚扰
- 未经许可发布他人的个人信息（如物理或电子邮件地址）
- 其他不道德或不专业的行为

## 责任与执行

项目维护者有责任澄清和执行我们社区的可接受行为标准。他们有权在认为适当的情况下删除、编辑或拒绝评论、提交、代码、维基编辑、问题和其他贡献，以及临时或永久禁止任何他们认为行为不当、威胁、冒犯或有害的贡献者。

### 范围

本行为准则适用于项目空间（包括但不限于 GitHub、讨论区、文档等），也适用于项目维护者在公共空间中代表项目或社区时的行为。

## 执行指南

### 处理违规行为

1. **识别问题**：任何社区成员都可以指出不当行为
2. **私下沟通**：优先通过私信沟通解决问题
3. **维护者介入**：如问题持续，维护者将介入调解
4. **采取行动**：必要时采取警告、临时禁言或永久禁言等措施

### 报告违规

如果你遭遇或目睹不当行为，或有任何其他担忧，请通过以下方式联系维护者：

- 邮件：your-email@example.com（请替换为实际邮箱）
- GitHub 私信：[@Omnithyn](https://github.com/Omnithyn)

所有投诉将被认真审查和调查，并以维护者认为适当的方式处理。维护者有义务对报告者信息保密。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org)，版本 2.1，可在 [https://www.contributor-covenant.org/version/2/1/code_of_conduct.html](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html) 找到。
